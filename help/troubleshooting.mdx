If you're facing issues with cantaê, you can try the following steps to resolve them.

## Log files

Check the log files for any errors. You can find the log files in the following locations:
- on Linux: ~/.config/\{app name}/logs/main.log
- on macOS: ~/Library/Logs/\{app name}/main.log
- on Windows: %USERPROFILE%\AppData\Roaming\\\{app name}\logs\main.log

## Firewalls

Sometimes firewalls may block cantaê from accessing the internet. You can try disabling the firewall temporarily to see if it resolves the issue.

## Antivirus

Antivirus software may sometimes block cantaê from running. You can try disabling the antivirus temporarily to see if it resolves the issue.

## Reinstall

If none of the above steps work, you can try reinstalling cantaê. You can download the latest version of cantaê from the [official website](https://cantae.app).
