## Install Cantaê

{/* ### macOS

1. Download the latest version of cantaê Desktop App from the [Cantaê website](https://cantae.app).

3. Drag and drop Cantaê Desktop App to the Applications folder.

### Windows

1. Download the latest version of Cantaê Desktop App from the [Cantaê website](https://cantae.app).

3. Follow the on-screen instructions to install Cantaê Desktop App.
